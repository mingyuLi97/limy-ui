# Button

::: card

### 介绍

按钮
:::

## API

::: card-prop

### Props

| 参数     | 说明                                                          | 类型        | 默认值  |
| -------- | ------------------------------------------------------------- | ----------- | ------- |
| type     | 类型, 可选值为：`default` `primary` `info` `warning` `danger` | _`String`_  | default |
| disabled | 是否禁用按钮                                                  | _`Boolean`_ | false   |
| shape    | 按钮形状, 可选值为：`square` `round` `default`;               | _`String`_  | default |

:::
