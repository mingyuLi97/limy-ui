## API

::: card

### Props

| 参数    | 说明                    | 类型                          | 默认值 | 必要的 |
| ------- | ----------------------- | ----------------------------- | ------ | ------ |
| tag     | 渲染的标签名            | `keyof HTMLElementTagNameMap` | div    | false  |
| justify | Flex 布局主轴对其方式   | `string`                      | start  | false  |
| align   | Flex 布局交叉轴对齐方式 | `string`                      | center | false  |

:::
